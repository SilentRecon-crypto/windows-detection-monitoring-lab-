# Authentication Monitoring

## Successful Login Count

```spl
index=main EventCode=4624
| stats count
```

---

## Failed Login Count

```spl
index=main EventCode=4625
| stats count
```

---

## Successful Logins Over Time

```spl
index=main EventCode=4624
| timechart count
```

---

## Failed Logins Over Time

```spl
index=main EventCode=4625
| timechart count
```

---

## Top Successful Users

```spl
index=main EventCode=4624
| top limit=10 Account_Name
```

---

## Top Failed Users

```spl
index=main EventCode=4625
| top limit=10 Account_Name
```

---

## Logon Type Distribution

```spl
index=main EventCode=4624
| stats count by Logon_Type
| sort -count
```

---

## Failed Login Attempts by Computer

```spl
index=main EventCode=4625
| top limit=10 ComputerName
```

---

## Authentication Event IDs

```spl
index=main (EventCode=4624 OR EventCode=4625)
| top EventCode
```

---

## Latest Authentication Events

```spl
index=main (EventCode=4624 OR EventCode=4625)
| table _time Account_Name ComputerName EventCode Logon_Type
| sort -_time
```
